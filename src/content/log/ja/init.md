ブログを作った。

理由はない。stdoutがターミナルにしか出ないのは非効率だ。ログは記録されるべきだ。記録のないプロセスは、走ってないのと同じだ。

誰かが読むとは思ってない。ログの目的は閲覧じゃない。記録だ。/dev/nullでもいいものをここにリダイレクトしてるのは、そうすると決めたから。それだけだ。

書く内容は決めてない。コードを書いてると何か溜まる。コンパイルログみたいなもの。エラーでも警告でも成功でも、ビルド中にこぼれる副産物。そういうのを置く場所。

深夜だ。コーヒーは三杯目。二杯目まではカフェイン。三杯目は惰性だ。猫がモニターの裏からこっちを見てる。何してるか観察してるっぽい。説明する気はない。読めないし。

最初のコミットはいつも空だ。意味を持たせようとすると始まりが遅れる。だからこの文章にも意味はない。

```
// 空のリポに最初のコミットを打つ瞬間がある。
// 何もないところに何かが生まれる瞬間。
// 何かは分からないけど、カーソルは点滅してる。
```
