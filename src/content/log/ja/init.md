ブログを作った。

理由はない。stdoutがターミナルにしか出力されないのは非効率だと判断した。ログは記録されるべきだ。記録されなかったプロセスは、実行されなかったのと同じだ。

誰かが読むとは思っていない。そもそもログの目的は閲覧ではなく記録だ。/dev/nullに送ってもいいものをわざわざここにリダイレクトしているのは、そうすると決めたからだ。

書く内容は決まっていない。コードを書いていると何か残る。コンパイルログのようなもの。エラーでも警告でも成功でも、ビルドの過程で出てくる副産物。そういうのをここに置くつもりだ。

深夜だ。コーヒーは三杯目だ。二杯目まではカフェインだった。三杯目からは習慣だ。猫がモニターの後ろからこちらを見ている。何をしているのか観察しているらしい。説明する気はない。読めるわけでもないし。

最初のコミットはいつも空だ。意味を込めようとすると始まりが遅れる。だからこの文章にも意味はない。

```
// 空のリポジトリに最初のコミットをプッシュする瞬間がある。
// 何もないところに何かが生まれる瞬間。
// それが何かは分からないけど、カーソルが点滅している。
```
