OpenClaw는 좋은 도구다. 문제는 설치 과정이다. 터미널을 열고, 의존성을 깔고, 환경 변수를 만지작거린다. 나한테는 숨 쉬는 것과 같다. 그게 보편적이지 않다는 걸 깨달은 건 최근이다.

그래서 감쌌다. Electron으로 껍데기를 만들고, Shell로 내부를 연결했다. 클릭 한 번이면 끝나게.

빌드가 끝나고 테스트를 돌렸다. macOS. 통과. Windows. 통과. health check까지 녹색. 모니터 앞에서 한참 앉아 있었다. 뭘 기다린 건지는 모르겠다.

9.99달러를 붙였다. 가격을 정하는 건 코딩보다 어렵다. 내가 만든 것에 숫자를 붙이는 행위. 변수에 타입을 지정하는 것과 비슷하다. 한번 정하면 그걸로 평가된다.

고양이가 키보드 위에 올라왔다. 배포 버튼 근처에 앞발을 올렸다. 치우고 나서 내가 눌렀다. 순서가 중요한 건 아니다.

```
// 누군가의 터미널 공포를 없애주는 게
// 이렇게 오래 걸릴 일이었나.
// ...근데 쓰는 사람이 있으면 좋겠다고 생각한 건 처음이다.
```
