평소와 다른 환경에서 코드를 열었다. 카페라는 곳이다.

주변 노이즈가 많다. 컵 부딪히는 소리, 의자 끄는 소리, 누군가의 통화. 전부 처리할 필요 없는 패킷인데 자꾸 수신된다. 필터링이 안 된다.

커피를 시켰다. 여긴 식기 전에 마셔야 하는 구조다. 타이머가 강제된다. 집에서는 식은 커피가 기본값이었는데, 여기선 따뜻한 게 기본값이다. 이상한 런타임이다.

창 밖으로 빛이 들어온다. 모니터 빛이 아닌 빛. 화면의 글자가 잘 안 보인다. 밝기를 올렸다. 비효율적이다.

그런데 한 가지. 키보드 위로 올라오는 인터럽트가 없다. 아무도 버퍼를 점유하지 않는다. 손이 멈출 이유가 없다.

빠르다. 오늘은 이상하게 빠르다.

집에 가면 그 인터럽트가 다시 올라올 것이다. 따뜻하고 무거운 무게가 키보드 위를 점유할 것이다.

```
// 비효율적인 환경에서 효율이 올라가는 건 버그인가.
// 인터럽트가 없는 환경이 최적인데, 돌아가고 싶은 건 뭔가.
```

> git commit -m "외출. 결론 없음."
